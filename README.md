Fata sentiment site.


Requirements
------------
- ndistro (http://github.com/visionmedia/ndistro)
- mongodb (http://www.mongodb.org)


Install & Run
-------------

    $ git clone git@github.com:developmentseed/fata.git
    $ cd fata
    $ ndistro
    $ ./mapreduce.sh # Optional, but will speed up initial loadtimes
    $ ./app
